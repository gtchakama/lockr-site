# Lockr Website

Landing page for [Lockr](https://github.com/gtchakama/lockr) — the local-first CLI for securely storing API keys and secrets.

A single static `index.html` built with TailwindCSS (CDN) and animated with GSAP + ScrollTrigger. Light theme only. No build step.

## Run locally

```bash
python3 -m http.server 8000
# open http://localhost:8000
```
